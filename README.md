> *Built at a hackathon. Now production-ready.*

An autonomous AI agent system that works 24/7 like a full-time employee. It monitors your Gmail, drafts replies in your writing style, manages accounting through Odoo, posts to social media, and handles WhatsApp‚Äîall with human-in-the-loop approval before any sensitive action.

**The Architecture:**
```
Perception (Watchers) ‚Üí Memory (Obsidian Vault) ‚Üí Reasoning (AI) ‚Üí Action (MCP Servers)
```

**What makes it different:**
- **Privacy-first**: All data lives locally in an Obsidian vault. Nothing leaves your machine.
- **Human-in-the-loop**: AI drafts, you approve. No rogue agents sending emails.
- **Production-ready**: Docker deployment, 90-day audit logs, comprehensive pytest suite.
- **Multi-channel**: Gmail, WhatsApp, LinkedIn, Twitter, Facebook, Instagram, Odoo accounting.

`Python` `Docker` `PostgreSQL` `OpenAI` `Gmail API` `Twilio` `Odoo` `MCP Servers`

### [Warden](https://github.com/DevDonzo/warden) ‚Äî Autonomous Security Orchestrator

> *Published on npm. Used in CI/CD pipelines.*

A multi-agent security system that patrols your GitHub repos, identifies vulnerabilities via Snyk, generates verified patches, runs your test suite, and submits professional PRs‚Äîwithout you lifting a finger.

**The Council of Agents:**
- **Watchman** (Scanner) ‚Äî Integrates with Snyk and npm audit to find vulnerabilities
- **Engineer** (Fixer) ‚Äî Creates fix branches, patches dependencies, verifies tests pass
- **Diplomat** (Reporter) ‚Äî Crafts semantic PRs with proper labels and assignees

Operates under a strict "Rules of Engagement" constitution: never force pushes, never merges without approval, never submits untested code.

`TypeScript` `Node.js` `Snyk` `Octokit` `GitHub API` `Jest`
